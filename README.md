# テスト駆動で学ぶFirestoreセキュリティルール 【データ検証編】
 
## パッケージインストール
```bash
npm install
```

## テストの実行
### Firestoreエミュレータの起動
```bash
npm run emu:start:firestore
```

### テストの実施
```bash
npm test-watch
```
